#Day05 - Advanced CI/CD and Infrastructure Provisioning
## Jenkinsfile - With Git Integration and Testing

```groovy
pipeline {
    agent any
    stages {
        stage('Clone Repository') {
            steps {
                git 'https://github.com/rushikeshsatwadhar/Login-Signup-Form'
            }
        }
        stage('Build') {
            steps {
                echo 'Building project from Git repository'
            }
        }
        stage('Test') {
            steps {
                echo 'Running unit tests'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Deploying application'
            }
        }
    }
}

## Ansible - Create User and Sect Permissions

- nam: Create a new user
  hosts: all
  become: yes
  tasks:
    - name: Create user
      user:
        name: rushi
        state: present
        shell: /bin/bash
    - nameL: Set permissions for the user
      file:
        path: /home/rushi
        owner: rushi
        group: rushi
        mode: '0755'
