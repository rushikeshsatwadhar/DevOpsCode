# Day01 - Basics

## Ansible Task - INstall Nginx on Rushi Server

Rewrite this YAML manually:

---
- name: Install Nginx on Rushi server
  hosts: all
  become: yes

  tasks:
    - name: Install nginx
      yum:
        name: nginx
        state: present

## Terraform Task - Create EC2 (c5.xlarge)

provider "aws" {
    region = "us-east-1"
}

resource "aws_instance" "RushiEC2" {

## Kubernetes Task - Create Nginx Pod

apiVersion: v1
kind: Pod
metadata:
   name: rushi-pod
spec:
  containers:
   - name: web
     images: nginx:latest
     ports:
       - containerPort: 80

# Git Integration

## Jenkins Pipeline - Checkout Git
You will learn how to **checkout a Git repository** and list files.

pipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'https://github.com/rushikeshsatwadhar/Login-Signup-Form'
            }
        }
        stage('List Files'){
            steps {
                sh 'ls -la'
            }
        }
    }
}

## GitLab CI - Checkout Git
You will learn how to **checkout a Git repository in GitLab CI**.

stages:
  - checkout

git-checkout:
  stage: checkout
  script:
    - git clone https://github.com/rushikeshsatwadhar/Login-Signup-Form
    - cd Login-Signup-Form
    - ls -la
