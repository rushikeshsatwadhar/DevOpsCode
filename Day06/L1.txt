# Debugging Pipelines
## Jenkins Pipeline - Debugging
You will learn how to **fix broken stages and handle errors**.

pipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'https://github.com/rushikeshsatwadhar/Login-Signup-Form'
            }
        } 
        stage('Build') {
            steps {
                sh '''
                mvn clean compile || { echo "Compile failed"; exit 1; } 
                '''
            }
        }
        stage('Test') {
            steps {
                sh '''
                mvn test || { echo "Tests failed"; exit 1; }
                '''
            }
        }
    }
}
