# Day02 - Services & Tags

## Ansible - Install & Enable Nginx

---
- name: INstall and start Nginx on Rushi
  hosts: all
  become: yes

  tasks:
    - name: Install nginx
      yum:
        name: nginx
        state: present

    - name: Enable nginx
      service:
        name: nginx
        state: started
        enabled: yes

## Kubernetes - Deployment with 2 Replicas

apiVersion: app/v1
kind: Deployment
metadata:
  name: rushi-deployment
spec:
  replicas: 2
  selector:
    matchLabels:
      app: rushi-app
  template:
    metadata:
      labels:
        app: rushi-app
    spec:
      containers:
        - name: app
          image: nginx
          ports:
            - containerPort: 80

# Maven Build
## Jenkins Pipeline - Maven Build
You will learn how to **compile, test, and package a Java project** using Maven.

pipeline {
    agent any
    tools {
        maven 'Maven-3.8.1'
        jdk 'JDK-17'
    }
    stages {
        stage('Checkout') {
            steps {
                git url: 'https://github.com/rushikeshsatwadhar/Login-Signup-Form'
            }
        }
        stage('Compile') {
            steps {
                sh 'mvn clean compile'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
        stage('Package') {
            steps {
                sh 'mvn package'
            }
        }
    }
}
